🆕 Что нового в 2.0

    Раздельное управление башней и корпусом
    Теперь поворот башни и движение — независимые действия (как в реальных танках).

    Новые команды: TURN_LEFT, TURN_RIGHT, TURN_AROUND, SYNC_BODY, SET_ANGLE, RELATIVE_TURN, IF_ANGLE, WHILE_NOT_FACING, SHOW_AIM.

    Классический режим (чекбокс) – обратная совместимость со старыми уровнями (команды LEFT/RIGHT снова работают как поворот+движение).

    Новые объекты: Магнит (притягивает танк, если башня смотрит на него), Замедляющее поле (удваивает время действий).

    Расширенный BFS – учитывает угол башни, условия и циклы с направлением.

    Генератор уровней – теперь умеет создавать уровни с раздельным управлением, бэкдорами и несколькими путями.

    Режим «Танковый инженер» (для 12+) – добавляет инерцию, отдачу, перегрев пушки.

🎮 Игровой процесс

Вы управляете танком на клетчатом поле. Задача – собрать звезду (цель). Программа составляется из команд, которые танк выполняет последовательно. Можно использовать циклы, условия, функции, объекты и параллелизм.

Основные изменения в 2.0:

    Поворот башни и движение разделены.

    Выстрел, толкание, использование инструментов происходят в направлении башни.

    Классический режим возвращает старую логику LEFT/RIGHT.

📜 Команды управления танком
Движение и поворот башни (новые)
Иконка	Команда	Действие
⬆️	MOVE_FORWARD	Ехать вперёд на 1 клетку (по направлению башни)
⬇️	MOVE_BACKWARD	Ехать назад на 1 клетку (кормой)
⬅️	TURN_LEFT	Повернуть башню влево на 90° (без движения)
➡️	TURN_RIGHT	Повернуть башню вправо на 90°
🔄	TURN_AROUND	Развернуть башню на 180°
🧭	SYNC_BODY	Развернуть корпус в сторону башни
⚙️	(Classic Mode)	Включает старые LEFT/RIGHT (поворот+движение)
📐	SET_ANGLE	Установить абсолютный угол башни (0–270)
🔁	RELATIVE_TURN	Повернуть на относительный угол (например, +45)
Условные и циклические конструкции для углов
Иконка	Команда	Действие
🧭	IF_ANGLE == value	Выполнить блок, если угол башни равен заданному
🔁	WHILE_NOT_FACING target	Повторять, пока башня не смотрит на цель
🎯	SHOW_AIM	Показать линию выстрела (Developer mode)

Остальные команды (инвентарь, инструменты, бой, функции, ООП, параллелизм) остались без изменений (см. предыдущую документацию).
🧱 Объекты на поле (новые)
Иконка	Название	Эффект
🧲	Магнит	Притягивает танк на 1 клетку, если башня повёрнута в его сторону
⏱️	Замедляющее поле	Все действия танка (движение, поворот) занимают в 2 раза больше времени

Остальные объекты (платформа, стена, яма, ключи, двери, конвейеры, телепорты, клей, клетка и т.д.) – без изменений.

⚙️ Генератор уровней (CLI)

Генератор создаёт уровни с поддержкой всех новых механик, включая бэкдоры.
bash

# Сгенерировать 500 уровней для мира Meadow с бэкдорами
npm run generate:levels -- --world meadow --count 500 --backdoors --backdoorCount 2

# Сгенерировать уровни для Volcano с раздельным управлением
npm run generate:levels -- --world volcano --start 2001 --count 500 --turretControl separate

Параметры:

    --world – meadow, ocean, clouds, fairy, volcano, arcade, bonus

    --count – количество уровней

    --start – начальный ID

    --difficulty – диапазон (1-100)

    --backdoors – разрешить бэкдоры

    --backdoorCount – 1-3

    --turretControl – separate (новый) / classic (старый)

    --seed – для воспроизводимости

Уровни сохраняются в public/levels/, обновляется manifest.json.
🔍 BFS-валидатор (с поддержкой башни)

BFS учитывает:

    Позицию танка, угол башни, направление корпуса.

    Инвентарь (ключи, кукуруза, ядра, инструменты).

    Монстров (приручение, отвлечение, уничтожение).

    Телепорты, конвейеры, пружины, кнопки, мосты, клетки, ловушки.

    Новое: условия IF_ANGLE, циклы WHILE_NOT_FACING.

    Новое: магниты и замедляющие поля (как замедление времени).

Валидатор используется:

    При генерации уровней – проверка, что уровень имеет решение.

    В игре – для показа подсказок (следующий шаг).

    На сервере Arcade – для проверки пользовательских уровней перед публикацией.

🏛 Техническая архитектура (сокращённо)
Компонент	Технология
Движок	Phaser 3.80+ (Canvas/WebGL)
Язык	TypeScript 5.9+ (strict)
Сборка	Vite 5.x
Мобильные обёртки	Capacitor 6.x
PWA	Workbox (Service Worker)
CLI генератора	Node.js + Commander + seedrandom
BFS-валидатор	Чистый TS, без зависимостей
Бэкенд Arcade	Express + SQLite (опционально)

Ключевые модули (src/):

    types/index.ts – все enum и интерфейсы (Command, TileType, …)

    modules/Player.ts – управление танком (позиция, башня, инвентарь)

    modules/execution/movement.ts – движение с учётом башни

    modules/execution/runner.ts – цикл выполнения программы

    modules/execution/conditions.ts – условия (включая IF_ANGLE)

    tools/generator/levelGenerator.ts – генерация уровней

    tools/generator/pathfinder.ts – BFS с инвентарём и башней
